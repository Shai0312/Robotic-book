# External ROS 2 Tutorials and Examples

## Overview

This section provides links to valuable external resources for learning more about Python-ROS integration and humanoid robotics. These resources complement the educational content in this module and provide additional depth and real-world examples.

## Official ROS 2 Resources

### ROS 2 Documentation
- [ROS 2 Documentation](https://docs.ros.org/en/humble/) - Official ROS 2 documentation for Humble Hawksbill
- [ROS 2 Tutorials](https://docs.ros.org/en/humble/Tutorials.html) - Step-by-step tutorials for ROS 2 concepts
- [rclpy API Documentation](https://docs.ros.org/en/humble/p/rclpy/) - Detailed API reference for Python ROS client library

### Core Concepts
- [ROS 2 Concepts](https://docs.ros.org/en/humble/Concepts/About-ROS-2-Concepts.html) - Understanding fundamental ROS 2 concepts
- [Nodes](https://docs.ros.org/en/humble/Concepts/About-ROS-2-Nodes.html) - Deep dive into ROS 2 nodes
- [Topics](https://docs.ros.org/en/humble/Concepts/About-Topics.html) - Understanding publish-subscribe communication
- [Services](https://docs.ros.org/en/humble/Concepts/About-Services.html) - Request-response communication patterns
