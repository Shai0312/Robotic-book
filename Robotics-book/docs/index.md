# Welcome to the ROS 2 Robot Control Module

This comprehensive educational module is designed for AI engineers and robotics students who want to understand how to connect AI agents to humanoid robot control systems.

## What You'll Learn

- **ROS 2 Core Concepts**: Understand nodes, topics, services, and messages
- **Python-ROS Integration**: Connect Python-based AI agents to robot control systems
- **Humanoid Modeling**: Create complete robot descriptions using URDF

## Getting Started

[Get started with the Quick Start Guide](./quickstart.md) to set up your environment, or browse the documentation using the sidebar to explore specific topics.

## Module Structure

This module is organized into three main chapters covering ROS 2 concepts, Python integration, and URDF modeling for humanoid robots.
